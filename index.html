What is Canopy?

Canopy is a sitemap toolkit.

It crawls a live website, maps the structure of every page it can reach, and generates a visual tree diagram — useful for understanding a site before a redesign, migration, or audit, and for sharing that structure with a client.

The workflow

🕷️ Crawler — point it at any URL and Canopy walks all reachable internal pages. Use the Workbench to filter out what you don't need before you start. When done, hit → build diagram.

🌳 Diagram — an interactive tree of the site laid out by section and depth. Pan and zoom to explore. Root, section nodes, and leaf pages are sized distinctly.

🍂 Export — download as PNG or SVG for presentations, CSV for spreadsheets, or JSON for developers.

A note on crawling

Browsers block cross-origin requests, so some sites will return CORS errors. Canopy marks these as blocked and continues. For full crawl coverage of external sites, a proxy is needed — this is planned as a future feature.
